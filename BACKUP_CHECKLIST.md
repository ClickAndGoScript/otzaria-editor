# ✅ רשימת בדיקות גיבוי

## 📅 פעם בשבוע (כל יום ראשון בבוקר)

1. [ ] היכנס לפאנל הניהול
2. [ ] לחץ על כפתור "גיבוי מלא" (ירוק)
3. [ ] שמור את הקובץ ב-Google Drive בתיקייה "Otzaria Backups"
4. [ ] שנה שם לקובץ: `backup-YYYY-MM-DD.json`

**זמן משוער**: 2 דקות

---

## 📅 פעם בחודש (ראשון בחודש)

1. [ ] הרץ בדיקת גודל: `node scripts/check-mongodb-size.js`
2. [ ] בדוק שהשימוש מתחת ל-50%
3. [ ] הורד גיבוי נוסף ושמור במחשב
4. [ ] מחק גיבויים ישנים מעל 3 חודשים

**זמן משוער**: 5 דקות

---

## 🔔 הגדר תזכורות

2. כל יום ראשון ב-10:00
3. הוסף קישור: https://otzariaeditor.vercel.app/admin

### טלפון:
1. הוסף תזכורת שבועית
2. "הורד גיבוי אוצריא"
3. כל יום ראשון

---

## 📊 מה לבדוק בגיבוי

פתח את הקובץ JSON ובדוק:
- `stats.totalUsers` - מספר משתמשים
- `stats.totalBooks` - מספר ספרים
- `stats.totalPages` - מספר עמודים
- `exportDate` - תאריך הגיבוי

אם המספרים נראים תקינים - הכל בסדר! ✅

---

## 🆘 במקרה חירום

אם משהו קרה לנתונים:

1. **אל תיבהל!** יש לך גיבויים
2. הורד את הגיבוי האחרון מ-Google Drive
3. צור קשר איתי (Kiro) לעזרה בשחזור
4. או פנה ל-MongoDB Support

## 📍 איפה הגיבויים?

- **Google Drive**: תיקייה "Otzaria Backups"
- **מחשב**: `C:\Users\[שם]\Downloads\otzaria-backups\`
- **MongoDB**: Atlas Dashboard → Backups
