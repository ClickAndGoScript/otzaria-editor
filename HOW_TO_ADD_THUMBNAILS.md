# איך להוסיף תמונות תצוגה מקדימה (Thumbnails)

## למה זה חשוב?

תמונות תצוגה מקדימה מאפשרות למשתמשים לראות את תוכן העמוד לפני שהם בוחרים לערוך אותו.

## מבנה התיקיות

```
public/
├── assets/
│   └── library/
│       └── בראשית.pdf
└── thumbnails/
    └── בראשית/
        ├── page-1.jpg
        ├── page-2.jpg
        ├── page-3.jpg
        └── ...
```

## שלב 1: המר PDF לתמונות

### אופציה 1: שימוש ב-ImageMagick (מומלץ)

```bash
# התקן ImageMagick
# Windows: https://imagemagick.org/script/download.php

# המר PDF לתמונות
magick convert -density 150 בראשית.pdf -quality 85 page-%d.jpg
```

### אופציה 2: שימוש ב-pdftoppm (Linux/Mac)

```bash
pdftoppm -jpeg -r 150 בראשית.pdf page
```

### אופציה 3: שימוש באתרים אונליין

- https://www.ilovepdf.com/pdf_to_jpg
- https://smallpdf.com/pdf-to-jpg

## שלב 2: ארגן את התמונות

1. צור תיקייה עם שם הספר (ללא .pdf):
   ```
   public/thumbnails/בראשית/
   ```

2. שנה שמות לפורמט: `page-1.jpg`, `page-2.jpg`, וכו'

3. ודא שמספר העמוד תואם למספר בקובץ

## שלב 3: רענן את האתר

המערכת תזהה אוטומטית את התמונות ותציג אותן!

---

## טיפים:

### איכות מומלצת:
- **רזולוציה:** 150 DPI
- **איכות:** 85%
- **פורמט:** JPG (קטן יותר מ-PNG)

### גודל קובץ:
- כל תמונה צריכה להיות בערך 50-150 KB
- אם גדולה מדי, הקטן את האיכות

### שמות קבצים:
- ✅ `page-1.jpg`, `page-2.jpg` (נכון)
- ❌ `page1.jpg`, `1.jpg` (לא יעבוד)

---

## דוגמה מלאה:

### יש לך: `public/assets/library/תלמוד/ברכות.pdf`

### צור:
```
public/thumbnails/תלמוד/ברכות/
├── page-1.jpg
├── page-2.jpg
├── page-3.jpg
└── ...
```

### המערכת תחפש אוטומטית:
```
/thumbnails/תלמוד/ברכות/page-{מספר}.jpg
```
